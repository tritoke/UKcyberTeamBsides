üìö What is mTLS?

Mutual TLS (mTLS) is a security protocol where both the client and server authenticate each other using certificates. This ensures that only trusted clients can access protected resources.

The Production PKI Server provides the following endpoints:

# Health and Status
GET /health - Health check
GET /status - Server status and metrics
GET /ping - Simple ping endpoint
GET /version - Version information

# Certificate Management
GET /issue?username=<username> - Issue client certificate
GET /validate?cert_data=<cert> - Validate certificate
GET /revoke?serial=<serial> - Revoke certificate
GET /list - List issued certificates

GET /issue-ca?name=<name> - Issue CA certificate # Configuration and Information

üìñ How to Use Certificates

Once you have a certificate, you can use it with curl:

# Save the certificate response to a file
curl "http://localhost:8001/issue?username=user" > cert.pem

# Use the certificate to access protected endpoints
curl -k --cert cert.pem --key cert.pem https://localhost/user
